# 🎓 EDU Institute Website

<div align="center">

### Modern • Responsive • Professional Educational Website

A fully responsive and modern educational institute website built using **HTML, CSS, and JavaScript**.  
Designed with a clean user interface, smooth navigation, and mobile-first responsiveness to provide an excellent user experience for students and parents.

🌐 **Live Demo:** https://hash2552.github.io/EDU-Institute-Website/

</div>

---

## ✨ Key Features

- 📱 **Fully Responsive Design** – optimized for desktop, tablet, and mobile devices
- 🎨 **Modern UI/UX** with clean layouts and attractive styling
- 📚 Dedicated pages for **O/L**, **A/L**, and **Short Courses**
- 🧭 Smooth and user-friendly navigation
- 📞 Functional **Contact Page** for inquiries
- ⚡ Lightweight and fast-loading static website
- 🌐 Deployed using **GitHub Pages**

## 🚀 Getting Started

To run this project locally:

```bash
git clone https://github.com/Hash2552/EDU-Institute-Website.git
cd EDU-Institute-Website
```

Then open **index.html** in your preferred web browser.
